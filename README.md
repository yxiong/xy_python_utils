xy_python_utils
===============

Python utilities by Ying Xiong.

Author: Ying Xiong.  
Created: Feb 04, 2013.

Usage
-----

Install prerequisites:

    pip install setuptools numpy matplotlib


Install the package in development mode:

    python setup.py develop


Run unit tests:

    cd xy_python_utils
    python -m unittest discover -p "*_test.py"
